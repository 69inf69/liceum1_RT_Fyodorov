# Training Planner

Автоз: [Федоров Арсений Федорович]

## Описание

Приложение для планирования и учёта тренировок с возможностью фильтрации по дате и типу тренировки. Данные сохраняются в JSON‑файл.

## Функциональность

* Добавление тренировок с указанием даты, типа и длительности.
* Фильтрация тренировок по типу и дате.
* Сохранение данных между запусками.
* Валидация ввода данных.
